# Carcas-counters public
This is a contamination tracking software for a slaughterhouse. 
It has the following functionality:
- Creation of a batch
- Switching between batches
- Tracking of different types of contamination in determined areas of a carcas(it is possible to increment or decrement contaminant count)
- Data synchronization(if multiple people are inspecting the same batch, then changes of one are immediately reflected on all other devices)
- View of all historic inspections
- View of report for previous day/days
- In report, ability to swithc between summary of a signle batch or reports by specific days
